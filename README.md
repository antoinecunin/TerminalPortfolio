# Terminal Portfolio

Interactive Unix-style terminal portfolio.

🌐 **[antoinecunin.fr](https://antoinecunin.fr)**

## Stack

- React 19 + TypeScript
- Zustand (state management)
- Vite (build)

## Available commands

### Navigation

| Command | Description |
|---------|-------------|
| `ls` | List directories (about, experience, projects...) |
| `cd` | Navigate through the portfolio |
| `cat` | Read file contents |
| `pwd` | Print current path |
| `tree` | Display full directory tree |
| `find` | Search files by name |
| `grep` | Search text within files |

### Portfolio

| Command | Description |
|---------|-------------|
| `whoami` | Personal information |
| `man` | Manual pages (experience, education, projects) |
| `finger` | Display testimonials |
| `env` | Environment variables and contact links |
| `export` | Change theme or language |
| `uname` | Technical stack information |
| `ssh` | Browse project source code |
| `help` | List available commands |

## Development

```bash
npm install
npm run dev
```
